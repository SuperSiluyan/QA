- Зарегистрируйся в github и пришли куратору задания свой логин 
- Куратор отправит приглашение которое нужно будет принять по почте
- клонируй локально проект https://github.com/SuperSiluyan/QA
- после клонирования через терминал создай ветку для этого репозитория, название можно сформировать так - "branch_{твоя фамилия латиницей}"

**Задание 2: Твой первый коммит**
- скачай себе на ПК любую понравившуюся картинку
- через терминал добавь эту картинку и создай коммит в ветке из задания №1 с сообщением “Домашнее задание для QA”
- запушь свой коммит в удаленный репозиторий
